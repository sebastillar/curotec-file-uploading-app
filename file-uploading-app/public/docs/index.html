APIs for managing files, versions and comments

Add New Version

Add a new version to an existing file.

Example request:
curl --request POST \
    "https://0684-190-64-82-98.ngrok-free.app/api/files/1/versions" \
    --header "Content-Type: multipart/form-data" \
    --header "Accept: application/json" \
    --form "file=@/tmp/phppmgztp" 
const url = new URL(
    "https://0684-190-64-82-98.ngrok-free.app/api/files/1/versions"
);

const headers = {
    "Content-Type": "multipart/form-data",
    "Accept": "application/json",
};

const body = new FormData();
body.append('file', document.querySelector('input[name="file"]').files[0]);

fetch(url, {
    method: "POST",
    headers,
    body,
}).then(response => response.json());

Get Latest Uploads

Retrieve the latest file uploads with pagination.

Example request:
curl --request GET \
    --get "https://0684-190-64-82-98.ngrok-free.app/api/files/latest?limit=15&page=1" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"limit\": 9,
    \"page\": 20
}"

const url = new URL(
    "https://0684-190-64-82-98.ngrok-free.app/api/files/latest"
);

const params = {
    "limit": "15",
    "page": "1",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "limit": 9,
    "page": 20
};

fetch(url, {
    method: "GET",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
